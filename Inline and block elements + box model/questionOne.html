Create a div with the following styles:

  • padding: 20px
  • margin: 10px
  • border: 2px solid black
  • width: 200px

What is the total width of the element (without box-sizing and with box-sizing: border-box)?

Answer: Without box-sizing: border-box, the total width = 200px (width) + 40px (padding) + 4px (border) = 244px. With box-sizing: border-box, the total width remains 200px as padding and border are included inside the defined width.

Try changing padding and border. How does it affect total width?

Answer: Without box-sizing, changing padding or border increases the total width beyond the specified 200px. With box-sizing: border-box, the box keeps its width fixed at 200px, adjusting the content space inside instead.
