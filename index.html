  1. Box Model

    • Box Model Basics & box-sizing Behavior

      This file demonstrates foundational CSS concepts related to the Box Model, including the roles of padding, margin, border, and width. It explores how total element size is calculated differently when using the default box-sizing: content-box versus box-sizing: border-box. The page includes styled elements for visual comparison and explanations that illustrate how changes in padding and borders affect an element’s total rendered size.

    • Nested Boxes with Different Padding, Margin, and Overflow Behavior

      This code demonstrates the behavior of nested div elements within a parent container, focusing on their padding, margin, and overflow properties. The parent div contains two child divs, each with different styles applied—one child has box-sizing: border-box, while the other does not. The margin overlaps between the divs, but no other content or spacing issues arise. The overflow: hidden property on the parent container hides any content that extends beyond its bounds, behaving like a magician’s hat where content appears and disappears.

  2. CSS Grid

    • Responsive CSS Grid Layout with Hover Effects

      This code creates a responsive layout using CSS Grid to arrange six content boxes inside a styled container. On smaller screens, the boxes stack vertically in a single column. On wider screens (minimum 40rem), the layout shifts to a 3-column, 2-row grid, with the first box (boxOne) spanning all three columns for emphasis. Each box has a unique background color, consistent padding, shadows, and a hover effect that scales it up slightly with smooth transitions for visual interactivity. The layout is clean, adaptable, and designed for readability and user engagement.

    • Responsive Grid Layout with Nested Grid and Area Template

      This HTML and CSS project creates a responsive layout using CSS Grid, featuring four main boxes arranged into designated grid areas: header, sidebar, content, and footer. Box 3 (boxThree) is a nested grid that holds three internal content sections (box 3.1, box 3.2, and box 3.3), adapting from a single column on smaller screens to a 3-column layout on wider screens. Each box includes hover effects for interactive scaling, custom shadows for depth, and consistent padding and border-radius for a polished UI. The entire layout is styled with a monospaced font and enhanced visual separation using distinct background colors.

    • Asymmetric Grid Layout with Span-Based Positioning

      This layout demonstrates advanced CSS Grid positioning with non-sequential grid areas. On smaller screens, all six boxes stack in a single column for mobile friendliness. Once the screen width exceeds 40rem, the layout shifts into a two-column grid where: Box 1 spans the full width of the top row. Box 2 stretches vertically across two rows in the left column. Box 3 and Box 5 span the entire row width independently. Box 4 and Box 6 are stacked vertically in the right column but separated by the grid row structure. This setup offers a dynamic, non-linear visual flow—perfect for magazine-style layouts, dashboards, or storytelling interfaces. Smooth scaling on hover and rounded edges help create a modern, interactive design feel.

  3. CSS Flex-Box

    • Flexbox Mini Portfolio – Responsive UI Showcase

      This project is a Flexbox-powered mini portfolio that demonstrates a variety of layout challenges and solutions using modern CSS techniques. Each section addresses a different common UI pattern, making it an excellent practice portfolio or template for beginner to intermediate front-end developers. It includes:

      1. Flex Navbar: A responsive navigation bar using flex and justify-content: space-evenly for balanced spacing.
      2. Responsive Card Grid: A flexible card layout that wraps on smaller screens and maintains clean spacing using flex-wrap.
      3. Login Button: A centered, interactive login box that scales up on hover for better UX.
      4. Sidebar Content Layout: A classic sidebar-content split using Flexbox, ideal for dashboards or blog layouts.
      5. Calculator Button Grid: A button grid mimicking calculator keys using nested Flexbox containers.
      6. Profile Card Column Layout: A styled profile card with image, description, and a follow button with hover animations.
      7. Gallery Layout: A responsive image gallery with hover-zoom animation using Flexbox and flex-wrap.
      Each component is styled with a consistent color palette, interactive hover effects, and smooth transitions to enhance the user experience. This portfolio emphasizes Flexbox fundamentals in real-world UI use cases.
